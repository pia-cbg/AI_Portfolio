# 📅 LLM Scheduler (일정 자동 등록 AI 모듈)

이 모듈은 자연어 일정을 LLM(예: GPT)을 이용해 파싱,  
실제 등록 가능한 데이터(캘린더 등)로 변환하는 AI 스케쥴러 엔진입니다.

---

## 🗂️ 폴더 및 파일 설명

### cli/
- 일정 관련 AI 기능을 커맨드라인에서 직접 실험/테스트하기 위한 CLI 진입점

### data_processing/
- **auto_date_generator.py**  
  배치 평가용 자동 질문(테스트 케이스) 생성기

### eval/
- **batch_cli.py**  
  여러 일정 쿼리를 일괄 평가하는 배치 평가 실행 스크립트
- **evaluator.py**  
  (미구현) 실제 평가 알고리즘 구현 예정

### prompts/
- LLM(예: GPT) 프롬프트 템플릿 관리

### models/
- **schedule_llm.py**  
  LLM이 받은 자연어 명령에서 일정 정보를 추출

### utils/
- **date_utils.py**  
  자연어 일정 → 표준 ISO 포맷 → 캘린더 등록용 데이터 변환 유틸
  (여러 모듈에서 재사용, 추후 통합 확장 목적)

### main.py
- (미구현) 오케스트레이션(통합 파이프라인)에서 import하여  
  전체 일정처리 체인을 제공하는 메인 인터페이스  
  (외부서비스, orchestration, API 등에서 호출 용도)
