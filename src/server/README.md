# 🚀 src/server (실서비스 연동/서버 엔트리포인트)

이 폴더는 실제 서비스 및 외부 연동(디스코드, 구글 캘린더 등)을 위한  
서버, 인증, API 관련 코드를 관리합니다.

---

## 📁 주요 파일 설명

- **gcal_api.py**  
  구글 캘린더 일정 등록 API 호출 함수

- **auth_manager.py**  
  구글 인증 및 토큰/자격증명 관리 함수

- **google_auth_server.py**  
  구글 인증(OAuth) 콜백 처리를 위한 Flask 기반 웹서버

---

## 💡 참고

- MVP(시범 단계)에서 실제 디스코드/캘린더 연동까지 개발 완료한 바 있으나,  
  현재는 소스코드만 유지하며 실제 연동/실행은 중지된 상태입니다.
- 추후 오케스트레이션 통합이 완료되면  
  본 폴더 내 코드를 실서비스 연동/외부 API 등의 목적으로 재사용할 예정입니다.
